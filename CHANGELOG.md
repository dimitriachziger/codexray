## [0.1.0] - 2026-07-23

### Added

- Exact token accounting and estimated visible-context attribution for Codex
  rollout files.
- Single- and multi-session reports in text and versioned JSON formats.
- Diagnostics for repeated, overlapping, and large tool output.
- Explicit snippet opt-in with bounded output and home-path redaction.
- CLI help, version output, doctor checks, and stable exit-code behavior.
- Strict JSON Schema for the public report contract.
- Node.js 22 and 24 CI, community documentation, and repository templates.
